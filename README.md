# 📖 Giới thiệu

**Chân Trời Số Việt Nam 2050** là một project infographic tương tác mô phỏng viễn cảnh phát triển công nghệ của Việt Nam trong tương lai.

Project tập trung vào:

- 🤖 Trí tuệ nhân tạo (AI)
- 📡 Điện tử – Viễn thông
- 🌐 Chuyển đổi số quốc gia
- 🏙 Thành phố thông minh
- 📈 Kinh tế số
- 🔐 An ninh mạng
- 🚗 Công nghệ tự động hóa

Mục tiêu của project là trình bày thông tin bằng giao diện trực quan, hiện đại và dễ tương tác.
